Support

Support is provided through our SourceForge page. It is recommended that you create a SorceForge account to submit bugs and feature requests, as having an account will allow us to contact you in the event further details are needed.

Report Bugs

The MOEA Framework is not bug-free, nor is any other software application, and reporting bugs to developers is the first step towards improving the reliability of software. Critical bugs will often be addressed within days.

  • Browse Bug Reports
  • Submit Bug

Request New Features

We welcome input from the user community when deciding if any new features or extensions should be developed. Developers wishing to contribute to this project may also browse the list of feature requests when looking for new development ideas.

  • Browse Feature Requests
  • Submit New Feature Request

Extended Support

For extended support options or other inquiries, please contact admin@moeaframework.org.
